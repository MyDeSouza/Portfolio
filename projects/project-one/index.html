Product Design & Systems Architecture

Cadence

An agentic workspace and orchestration protocol that unifies fragmented software ecosystems into a single, cohesive layer.

View case study

Overview

The Problem

Modern knowledge workers waste hours acting as human middleware — manually copying data, tracking tasks across siloed apps, and constantly fighting interface fatigue. Instead of empowering focus, our software stack demands non-stop administrative management, fracturing user attention and draining productivity.

Process

How we approached it

To solve application fragmentation, the design required looking past surface-level UI patches and re-engineering the workflow layer from the ground up, balancing deep digital systems with tactile physical hardware.

End-to-End Workflow Mapping

Analyzed complex, cross-platform user journeys to identify key friction points. Designed a post-GUI orchestration architecture that predictively automates repetitive data-routing and task handoffs.

Prototyping Across Mediums

Balanced digital interface design with physical product development. Manufactured custom hardware touchpoints using laser-etched, heat-bent frosted acrylic to explore how ambient system states can be communicated cleanly outside of a crowded screen.

Minimalist Interaction Philosophy

Stripped away UI noise to create an "in-the-background" user experience. Every feature was optimised to reduce cognitive load, ensuring the system operates autonomously until high-leverage user intent is required.

Outcome

Results

Cadence delivers a functional blueprint for the future of intentional, AI-assisted productivity.

Unified Workspace Ecosystem

Successfully built a scalable framework that connects disparate applications into a single, fluid operational stream.

Frictionless Execution

Eliminated the administrative drag of multitasking, allowing cross-functional users to focus entirely on execution rather than tool management.

Cross-Disciplinary Validation

Delivered a comprehensive 76-page documentation book alongside high-fidelity physical models, demonstrating a complete product vision ready for digital development and physical manufacturing.
